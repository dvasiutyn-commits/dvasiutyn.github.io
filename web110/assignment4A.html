The first networking activity I completed was Internet Reference Models. In this lesson I learned about the OSI Model and the TCP/IP Model. The OSI model contains seven layers including Physical, Data Link, Network, Transport, Session, Presentation, and Application. I learned that each layer has a specific role in network communication. The TCP/IP model is the practical model used on the Internet and includes the Application, Transport, Internet, and Host-to-Network layers. I learned how data moves through these layers when devices communicate over a network. Understanding these models helped me see how information travels from one computer to another and how different networking technologies work together. I can apply this knowledge in my future career in technology because understanding network layers helps with troubleshooting network problems, configuring systems, and understanding how websites and online services communicate over the Internet.

The second networking activity I completed was Internet Protocols. In this lesson I learned about several important protocols including TCP, IP, UDP, FTP, TFTP, Telnet, and HTTP. I learned that TCP provides reliable communication and ensures that data arrives correctly, while UDP provides faster communication without guaranteeing delivery. I also learned that IP is responsible for routing packets between devices across networks. FTP and TFTP are used for file transfers, while HTTP is the protocol used for communication between web browsers and web servers. Learning about these protocols helped me understand how different Internet services operate and why different protocols are used for different purposes. I can apply this knowledge in my future environment by better understanding how websites, applications, and networks function, which will help me as I continue learning web development and computer technology.
