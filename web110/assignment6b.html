Assignment 6B - Web Accessibility

In this lesson, I learned that web accessibility is important because it allows everyone, including people with disabilities, to use a website more easily. I learned that images should always include meaningful alternative text so screen readers can describe them. I also learned that good color contrast and valid HTML improve readability and accessibility for all users. These concepts will help me create more professional websites that follow accessibility standards and provide a better experience for every visitor. I will use these practices in my future web development projects to make my websites more inclusive and user-friendly.
