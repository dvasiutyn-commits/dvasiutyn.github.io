The first activity I completed focused on web browsers and their features. I learned that web browsers are software applications used to access and display information on the Internet. During this activity, I explored how browsers such as Google Chrome, Microsoft Edge, Mozilla Firefox, and Safari help users navigate websites, save bookmarks, manage downloads, and improve online productivity. I also learned about browser security features, including private browsing, password management, and software updates. These concepts helped me understand how important web browsers are for safely accessing information online. I can apply this knowledge by using browser settings to improve security, organize educational resources, and increase efficiency while completing school assignments and personal projects. In the future, understanding browser functionality will help me work more effectively online and troubleshoot common browsing issues.

The second activity focused on HTML and Internet fundamentals. I learned that HTML is the standard markup language used to create web pages and that it provides the structure of website content. I became familiar with common HTML elements such as headings, paragraphs, images, links, and page titles. I also learned how web browsers interpret HTML code and display it as a web page. This activity helped me understand the relationship between HTML, CSS, and JavaScript in web development. I can apply these concepts by creating my own web pages and improving my understanding of website design. In my future career, HTML knowledge will provide a strong foundation for learning more advanced web development technologies and building professional websites for businesses and personal projects.
