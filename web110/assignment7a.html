Final Project Topic

Building a Personal Website with HTML and CSS

Objectives and Scope

The objective of my final project is to demonstrate what I have learned in this course by creating a personal website using HTML and CSS. My presentation will explain how web pages are built, how navigation works, how external style sheets improve consistency, and how images and accessibility features make websites more professional and user-friendly. The project will include examples from my previous assignments and will show how these concepts work together to create a complete website. The scope of this project is limited to the topics covered during this course, including HTML, CSS, multimedia, hyperlinks, and accessibility.

Project Schedule (Gantt Chart)

Reflection

Working on this project planning assignment helped me understand the importance of organizing a project before starting the actual work. Creating a Gantt chart showed me how to break a large project into smaller tasks and assign realistic deadlines to each step. I learned that planning ahead helps reduce stress, keeps a project on schedule, and makes it easier to track progress. These project management skills will be valuable in future college assignments as well as in professional software development, where planning, scheduling, and meeting deadlines are essential for successful teamwork and project completion.
